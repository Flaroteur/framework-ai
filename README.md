# framework-ai

Обоснование выбора метрик:

Классификация
Recall (класс 1) — главная метрика: нужно найти ВСЕХ с риском выгорания
Precision — контроль ложных тревог (false positives)
F1-weighted — учет несбалансированности (6.45% положительных)
ROC-AUC — оценка способности модели ранжировать риски

Регрессия (Walmart Sales):
MAE — интерпретируемая ошибка в деньгах, устойчива к выбросам
RMSE — штраф за большие ошибки (важно для финансового планирования)
R² — доля объясненной дисперсии продаж
